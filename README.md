# He Warned You

**He Warned You** is a Minecraft Forge horror mod for **1.21.4**.

Something is watching you from behind.

At first, everything may seem normal. Then strange music can start playing. A warning may appear on your screen. If you turn around when you should not, you will see the entity… and the chase begins.

## Features

- Random creepy ambient music
- Sudden warning events
- Custom horror entity
- Dialogue with a corrupted mysterious voice
- 30-second chase sequence
- Red screen flashes and camera shake
- Player gets kicked if the entity catches them
- The entity remembers if you were caught

## Warning

This mod is made to scare the player.  
Test it in a copied world first.

## Version

Minecraft: **1.21.4**  
Mod Loader: **Forge**
